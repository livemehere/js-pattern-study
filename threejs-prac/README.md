# THREE-JS

[fundamental](https://threejs.org/manual/#ko/fundamentals)

## 나만의 정의

- 출발점은 Scene(장면) 이다.

## 용어 정리

- Geometry : 형상
- Material : 재료
- Mesh : 특정 입자의 크기를 분류할 수 있는 망(입자라고 생각하면 될듯)
- 종횡비 : `종` 분의 `횡` = `횡/종` = `가로/세로` = `너비/높이`


### Geometry

- `Geometry` 는 `뼈대`이다.
- `Geometry` 는 `BufferGeometry` 로 만들어진다.
